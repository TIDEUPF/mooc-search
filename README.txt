Software necessari:
- Python 3.6

Pasos:
- A traves d'un terminal (Powershell si es des de Windows) entra a la carpeta "beautifulsoup4-4.6.0".
- Executa "python setup.py install"
- Torna a la carpeta anterior i executa "python server.py"
- Ja pots accedir a través de localhost:8000/index2OK.html a la versió final
